# RPA Itaú (ITUB4)

Projeto de automação desenvolvido em **JavaScript (Node.js)** para coletar
dados financeiros do ativo **ITUB4** no site Investidor10 e gerar
um relatório em **Excel**.

##  Tecnologias
- Node.js
- Puppeteer
- XLSX

##  Como executar

1. Instale as dependências:
```bash
npm install

Execute o script:
node rpa_itau.js

Resultado:

O script gera um arquivo chamado:
relatorio_itau.xlsx
